راهنمای تغییر URL دستیار هوشمند

آموزش تغییر آدرس چت بات در دو نسخه مختلف

مقدمه

این راهنما به شما کمک می‌کند تا آدرس URL چت بات را در دو نسخه مختلف تغییر دهید. هر دو نسخه دارای عملکرد یکسان اما با روش‌های نمایش متفاوت هستند.

نمونه فایل های های HTML در فولدر SRC قراردارند.

"rayabot_chatpanel.html" : نمونه کد برای باز شدن پنل چت در صفحه جاری

"rayabot_newTab.html" : نمونه کد برای بازشدن پنل چت در تب جدید

در این راهنما فرض شده است آدرس دستیار هوشمند شما https://rayabot.ir/rayabot می باشد.

توجه: برای تغییر آدرس چت بات، باید فایل‌های HTML مربوطه را ویرایش کرده و آدرس جدید را جایگزین آدرس فعلی کنید.

نسخه ۱: چت بات در پنل داخلی (rayabot_chatpanel.html)

در این نسخه، چت بات در یک پنل داخلی در همان صفحه نمایش داده می‌شود. برای تغییر آدرس، باید آدرس iframe را ویرایش کنید.

rayabot_chatpanel.html 
<div id="chatbot-container">
    <button id="chatbot-close">×</button>
    <iframe id="chatbot-iframe" src="https://rayabot.ir/rayabot"></iframe>
</div>

مراحل تغییر URL:

فایل rayabot_chatpanel.html را در یک ویرایشگر متن باز کنید.
خط مربوط به iframe چت بات را پیدا کنید (خط ۸۶ در کد اصلی).
آدرس https://rayabot.ir/rayabot را با آدرس جدید جایگزین کنید.
فایل را ذخیره کرده و تغییرات را تست کنید.

نسخه ۲: چت بات در تب جدید (rayabot_newtab.html)

در این نسخه، با کلیک روی دکمه چت، یک تب جدید باز شده و چت بات نمایش داده می‌شود. برای تغییر آدرس، باید آدرس پنجره جدید را ویرایش کنید.

rayabot_newtab.html 
// باز کردن چت بات در تب جدید هنگام کلیک روی دکمه
chatbotBtn.addEventListener('click', () => {
    window.open('https://rayabot.ir/rayabot', '_blank');
    chatbotBubble.style.display = 'none';
});

مراحل تغییر URL:

فایل rayabot_newtab.html را در یک ویرایشگر متن باز کنید.
بخش اسکریپت مربوط به باز کردن تب جدید را پیدا کنید (خط ۷۴ در کد اصلی).
آدرس https://rayabot.ir/rayabot را با آدرس جدید جایگزین کنید.
فایل را ذخیره کرده و تغییرات را تست کنید.

کدهای کامل برای

کد کامل نسخه پنل داخلی:

rayabot_chatpanel.html - بخش iframe 
<div id="chatbot-container">
    <button id="chatbot-close">×</button>
    <iframe id="chatbot-iframe" src="https://rayabot.ir/rayabot"></iframe>
</div>

کد کامل نسخه تب جدید:

rayabot_newtab.html - بخش اسکریپت 
<script>
    const chatbotBtn = document.getElementById('chatbot-button');
    const chatbotBubble = document.getElementById('chatbot-bubble');

    // باز کردن چت بات در تب جدید هنگام کلیک روی دکمه
    chatbotBtn.addEventListener('click', () => {
        window.open('https://rayabot.ir/rayabot', '_blank');
        chatbotBubble.style.display = 'none';
    });

    // نمایش پیام خوشامد طبق بازدیدهای اول و بعدی
    window.addEventListener('load', () => {
        if (!localStorage.getItem('chatbotOpenedBefore')) {
            setTimeout(() => {
                chatbotBubble.style.display = 'block';
                localStorage.setItem('chatbotOpenedBefore', 'true');
            }, 5000);
        } else {
            setTimeout(() => {
                chatbotBubble.style.display = 'block';
            }, 3000);
        }
    });
</script>

نکات مهم

اطمینان حاصل کنید که آدرس جدید با پروتکل صحیح (http یا https) نوشته شده است.
پس از تغییر آدرس، حتماً عملکرد چت بات را تست کنید.
اگر از نسخه پنل داخلی استفاده می‌کنید، مطمئن شوید که آدرس جدید اجازه نمایش در iframe را می‌دهد.
در صورت بروز هرگونه مشکل، تنظیمات CORS آدرس جدید را بررسی کنید.
